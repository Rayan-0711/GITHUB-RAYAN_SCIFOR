1. what is HTML?
sol: HTML stands for Hyper Text Markup Language. It is a web development language which is used to create webpages.
It uses various tags and attributes to define elements like text, links, images, and multimedia.
It starts with a <!DOCTYPE> declaration followed by the html, head, and body sections.
basic html format:
<DOCTYPE html>
<head>
    <title>Basic HTML Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
</body>
</html>


2. what are HTML Tags?
sol: HTML tags are used to create HTML document, like it is used to define and structure elements on a webpage. They are enclosed in angle brackets (< >).
Normally they come In pairs as a opening tag and a closing tag that is called paired tag.
for ex: <body> </body>, <html> </html>
there is also another type of tag that is called non paired tags or singular tags, these are the tags which do not require a closing tag.
for ex: <img>, <br>
